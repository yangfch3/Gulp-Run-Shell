# Gulp Run Shell

1. 克隆仓库
    ```
    git clone git@github.com:yangfch3/Gulp-Run-Shell.git
    ```
2. 进入目录，安装依赖库
    ```
    npm install
    ```
3. 全局安装 `electron-packager`
    ```
    npm install -g electron-packager
    ```
4. 试运行
    ```
    npm run start
    ```
5. 打包
    ```bash
    # macOS
    npm run build:mac

    # windows
    npm run build:win
    ```
